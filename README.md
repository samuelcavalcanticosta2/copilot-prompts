# Copilot Prompts - Projeto DIO

## 📚 Sobre o Projeto
Este repositório contém os agentes de IA para orientação de carreira em tecnologia, desenvolvido no bootcamp da DIO em parceria com Caixa e Microsoft.

---

## 🧩 Estrutura
- **Agent 1**: Entrevistador de Carreira  
- **Agent 2**: Planejador de Carreiras  
- **Projeto Técnico**: Construção de uma SPA responsiva para pequenas e médias empresas

---

## 💻 Projeto Técnico
SPA com as seguintes seções:
- Header
- Navigation
- Home
- Sobre
- Serviços
- Depoimentos
- Contato
- Footer

Tecnologias sugeridas: React, Angular ou Vue.js.

---

## 📋 Pré-requisitos
- Lógica de programação (básico)  
- JavaScript (básico)  
- Gerenciamento de pacotes (básico)  

---

## 🎯 Objetivos
Após concluir:
- Entrevistar e sugerir carreiras com Agent 1  
- Montar roadmap personalizado com Agent 2  
- Construir SPA responsiva como projeto prático
